# Event Scheduler System using Threaded Binary Tree
## Overview
This project implements an Event Scheduler System using a Threaded Binary Tree (TBT) to efficiently manage event related operations.

## Features
- Insert events chronologically
- Display events in sorted order
- Search events by name
- Update event details
- Delete events
- Find the next upcoming event

## Technologies Used 
- C++
- Data Structures and Algorithms

## Data Structure Used
Threaded Binary Tree

## Learning Outcomes
- Practical implementation of Threaded Binary Tree
- Tree traversal operations
- Event scheduling logic
- Dynamic node management
